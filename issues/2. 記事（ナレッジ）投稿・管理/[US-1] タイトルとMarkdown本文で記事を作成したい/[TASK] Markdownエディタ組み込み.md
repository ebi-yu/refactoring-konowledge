## 概要

Markdownエディタコンポーネント（リアルタイムプレビュー付き）の組み込み。

## 親 STORY

## 作業内容

- [ ] Markdown エディタライブラリ（CodeMirror / Monaco Editor 等）を選定・導入する
- [ ] エディタとプレビューを左右に並べたスプリットビューコンポーネントを実装する
- [ ] 入力に合わせてリアルタイムでプレビューが更新される処理を実装する
- [ ] ツールバー（太字・斜体・リンク・コードブロック挿入ボタン）を実装する
- [ ] 全画面エディタモードに切り替えられるようにする

## 技術メモ

- プレビューには Markdown パーサー（marked.js / remark）と連携する
- エディタはモバイル対応も考慮した実装にする
