## 概要

管理者ゼロの状態でのみ呼び出せる初回管理者作成 API エンドポイントの実装。

## 親 STORY

- STORY: [US-2] 初回セットアップスクリプトで最初の管理者アカウントを作成したい

## 作業内容

- [ ] `POST /api/setup/admin` エンドポイントを実装する
- [ ] 管理者ユーザーが既に存在する場合は `409 Conflict` を返す
- [ ] リクエストボディで OAuth プロバイダーとアカウント識別子を受け取り、admin ロールを付与する
- [ ] エンドポイントはセットアップスクリプトからのみ呼び出し可能なよう、ローカルネットワーク制限またはシークレットトークンで保護する
- [ ] ロール付与後に成功レスポンス（201）を返す

## 技術メモ

- エンドポイントは初回セットアップ後に無効化（または環境変数フラグで制御）することを検討する
- セットアップトークンは `SETUP_SECRET` 環境変数で管理し、本番では強力なランダム文字列を設定する
