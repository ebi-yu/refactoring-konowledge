| 項目                   | 内容                                                                     |
| ---------------------- | ------------------------------------------------------------------------ |
| **Who**（誰が）        | 開発者                                                                   |

| **Why**（なぜ・価値）  | 普段使っているエディタで書いた記事をそのままコマンド一発で投稿したいから |

**ストーリー文（上記を文章化）:**

> **開発者** として、**CLIツールで記事の作成・投稿・編集をターミナルから操作したい**。なぜなら **普段使っているエディタで書いた記事をそのままコマンド一発で投稿したい** からだ。

## 背景・補足

Ph2スコープ。`knowledge` CLIコマンドを提供し、Markdownファイルをそのまま投稿・編集できるようにする。認証設定をCLIで行い、REST APIと連携する。
