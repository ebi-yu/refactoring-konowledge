## 概要

セキュリティのため埋め込みを許可するドメインのホワイトリスト設定と検証処理の実装。

## 親 Story

- Story: (US-6) 外部コンテンツをiframeで埋め込みたい

## 作業内容

- [ ] 許可ドメインのホワイトリスト設定ファイル（または環境変数）を作成する
- [ ] URL のドメインをホワイトリストと照合する検証関数を実装する
- [ ] ホワイトリスト外のドメインはテキストリンクとして表示する
- [ ] 管理者が管理画面からホワイトリストを設定・更新できる機能を実装する
- [ ] ホワイトリストの変更を反映するためのキャッシュクリア処理を実装する

## 技術メモ

- 初期ホワイトリスト：`youtube.com`, `youtu.be`, `speakerdeck.com`, `slideshare.net`, `figma.com`
- URL パースには `URL` API を使用してサブドメインも考慮した照合を行う
