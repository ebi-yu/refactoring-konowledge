**ストーリー文（上記を文章化）:**

> **閲覧者** として、**記事内のURLがOGPカードとして表示されているのを見たい**。なぜなら **生URLだけより内容が一目でわかってリンクを踏む前に何の記事かわかるから** だからだ。

## 背景・補足

URL だけの行を検出し、そのサイトの OGP メタデータ（タイトル・説明・OGP画像）を取得してカード形式で表示する。メタデータの取得はサーバーサイドで行い CORS 問題を回避する。

## 受け入れ基準

- [ ] **Given** URLのみの行を含む記事  
       **When** 詳細ページを開く  
       **Then** タイトル・説明・OGP画像を含むカードが表示される

- [ ] **Given** OGPメタデータのないURL  
       **When** カード取得する  
       **Then** フォールバック表示（URLのみ）になる
