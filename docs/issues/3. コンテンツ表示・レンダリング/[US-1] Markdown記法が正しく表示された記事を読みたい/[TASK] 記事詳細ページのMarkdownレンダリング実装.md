## 作業内容

- [ ] `MarkdownRenderer` コンポーネントを実装する（`body` プロパティで Markdown 文字列を受け取る）
- [ ] コンポーネント内で Markdown パーサーを使って HTML に変換して表示する
- [ ] コードブロック・テーブル・チェックボックスリスト等が正しく表示されることを確認する

## 技術メモ

- `v-html` を使用する際は必ずサニタイズ済みの HTML を渡す
- 記事本文のスタイルは専用 CSS クラス（例: `.markdown-body`）で管理する
