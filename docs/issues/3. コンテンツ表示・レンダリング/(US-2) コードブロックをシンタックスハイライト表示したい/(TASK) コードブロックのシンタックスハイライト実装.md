## 概要

Markdown レンダラーと連携してコードブロックに自動でシンタックスハイライトを適用する実装。

## 親 Story

- Story: (US-2) コードブロックをシンタックスハイライト表示したい

## 作業内容

- [ ] Markdown パーサーのカスタムコードブロックレンダラーにシンタックスハイライトを統合する
- [ ] 言語指定のないコードブロックは自動言語検出またはプレーンテキストで表示する
- [ ] コードブロックに言語ラベルとコピーボタンを付与する
- [ ] ハイライトのテーマを CSS 変数で管理してダークモード対応する

## 技術メモ

- `marked.js` の場合 `renderer.code` をカスタマイズして `hljs.highlight(code, { language })` を使用する
- Shiki の場合 `remark-shiki` プラグインで統合する
