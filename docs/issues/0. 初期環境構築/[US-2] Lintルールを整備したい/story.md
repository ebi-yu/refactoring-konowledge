## 概要

Oxlint の設定ファイル（`.oxlintrc.json`）を作成し、プロジェクト固有のルールを定義する。Vue ファイルの記述規約（Composition API 強制・Options API 禁止など）を lint で自動検出できる状態にする。

## 親 EPIC

- EPIC: (0) 初期環境構築

## ユーザーストーリー

**開発者** として、**lint がプロジェクトの記述規約を自動検出できる状態** にしたい。なぜなら **コードレビューで人手による規約チェックが不要になり、実装の一貫性を保てるから**。
