## 概要

Panda CSS のデザイントークン（カラー・タイポグラフィ・スペーシング・シャドウ・ボーダー半径）を `panda.config.ts` に定義し、全エピックで一貫したビジュアルを保てる基盤を作る。

## 親 EPIC

- EPIC: (0) 初期環境構築

## ユーザーストーリー

**開発者** として、**デザイントークンが一箇所に集約されている状態** にしたい。なぜなら **各エピックで UI を実装するときにカラーコードや余白サイズをバラバラに書くと一貫性が壊れ、後からデザインを直すコストが跳ね上がるから**。
