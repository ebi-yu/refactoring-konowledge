## 概要

トレンド記事一覧をバッジ（🔥など）付きで表示する UI コンポーネントの実装。

## 親 Story

- Story: (US-3) 勢い記事を確認したい

## 作業内容

- [ ] トレンド記事一覧コンポーネントを実装する（ランキング順位・🔥バッジ・タイトル・閲覧数表示）
- [ ] 「24時間」「7日間」のタブ切り替えを実装する
- [ ] トップページまたは `/trending` ページにトレンドセクションを配置する
- [ ] ランキング上位3件には金・銀・銅のバッジを表示する
- [ ] 定期的（例: 5分ごと）にトレンドデータを自動更新する処理を実装する

## 技術メモ

- タブ切り替えは `?period=24h` クエリパラメータで状態管理する
- トレンドデータの自動更新には `setInterval` またはサーバー送信イベント（SSE）を使用する

- [ ] セルフレビュー済み（lint / format エラーなし）
