## 概要

GET /api/search エンドポイントの実装（PostgreSQL 全文検索または Meilisearch 等を使ったタイトル・本文の横断検索）。

## 親 Story

- Story: (US-1) キーワードで全文検索したい

## 作業内容

- [ ] `GET /api/search?q=<keyword>` エンドポイントを実装する
- [ ] タイトル・本文・タグを対象にした全文検索クエリを実装する
- [ ] ログイン状態に応じた visibility フィルタリングを実装する（private 記事は本人のみ、login_only は認証済みのみ）
- [ ] 検索結果にキーワードのハイライト（スニペット）を含める
- [ ] ページネーションを実装する

## 技術メモ

- PostgreSQL の場合 `tsvector` / `tsquery` + `ts_rank` でスコアリングする
- 外部検索エンジン（Meilisearch / Typesense）を使う場合はインデックス同期処理も実装する
