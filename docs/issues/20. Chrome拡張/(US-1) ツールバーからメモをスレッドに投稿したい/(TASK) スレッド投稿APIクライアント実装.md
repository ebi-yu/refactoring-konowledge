## 概要

Chrome拡張からサーバーのスレッド投稿APIを呼び出すクライアント実装（認証トークン管理含む）。

## 親 Story

- Story: (US-1) ツールバーからメモをスレッドに投稿したい

## 作業内容

- [ ] サーバーのスレッド投稿APIを呼び出すAPIクライアント関数の実装
- [ ] `chrome.storage.local` からAPIトークンを取得してAuthorizationヘッダーに設定する処理
- [ ] 投稿成功時のポップアップ内フィードバック表示（「投稿しました」等）
- [ ] APIエラー時のエラーメッセージ表示（認証エラー・ネットワークエラー等）

## 技術メモ

- Background Service Workerで `fetch` APIを使ってサーバーにリクエストを送る
- CORSポリシーでChrome拡張からのリクエストを許可する設定が必要
- トークン期限切れ時は再ログインを促すメッセージを表示する
