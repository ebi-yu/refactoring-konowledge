## 概要

クラウドストレージのバケット内容をローカルに同期するバックアップスクリプトの作成。

## 親 Story

- Story: (US-5) バックアップ・リストアを実行したい

## 作業内容

- [ ] `scripts/backup-storage.sh` スクリプトの作成（aws s3 syncによるバケット同期）
- [ ] `scripts/restore-storage.sh` スクリプトの作成（ローカルからバケットへのリストア）
- [ ] バックアップ先ディレクトリの設定（環境変数 `BACKUP_DIR`）
- [ ] MinIO / R2 / S3 それぞれへの対応（エンドポイント設定の切り替え）

## 技術メモ

- AWS CLI の `aws s3 sync` コマンドを使用（MinIO・R2もエンドポイント指定で対応可能）
- `--delete` オプションで削除済みファイルもバックアップ先から削除して同期を保つ
- 増分バックアップになるため初回以降は差分のみ転送される
