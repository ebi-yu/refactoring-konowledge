# [EPIC] ゲーミフィケーション

## 概要

投稿・絵文字リアクション獲得などの活動にポイントを付与し、バッジや実績・ランキングで貢献を可視化する機能群。
**全機能が Ph2 スコープ。** Ph1 のコア機能が安定した後に追加する。

## EPIC ストーリー

**投稿者** として、**記事を書いたりコメントするたびにポイントが貯まり、バッジやランキングで自分の貢献が認められるのを感じたい**。なぜなら **継続的に投稿するモチベーションになるし、チームに貢献しているという実感が得られるから**。

## スコープ（STORY 一覧）

> 優先度: `Must` = リリース必須 / `Should` = できれば含める / `Could` = 余裕があれば
> **⚠️ このEPICは Ph2 スコープのため Ph1 リリースには含まれない**

| 優先度   | #   | ストーリー（一文）                                                                                                                                                   | Issue |
| -------- | --- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----- |
| `Must`   | 1   | 投稿者として、記事投稿や絵文字リアクション獲得などの活動でポイントが加算されるのを確認したい。なぜなら自分の貢献が数値で見えると継続して投稿するやる気が上がるから。 | -     |
| `Must`   | 2   | 投稿者として、特定の条件を達成したときにバッジや実績を獲得したい。なぜなら節目を祝ってもらえる感覚がモチベーションの維持につながるから。                             | -     |
| `Should` | 3   | 閲覧者として、ポイント上位ユーザーのランキングを見たい。なぜなら活発なメンバーを知ることでチームの盛り上がりを感じたいから。                                         | -     |

## スコープ外

- Ph1 では本 EPIC の全機能が無効

## 受け入れ基準（EPIC 完了の定義）

- [ ] 配下のすべての `Must` STORY が Done になっている
- [ ] ポイント付与・バッジ獲得・ランキング表示の E2E シナリオが通過している
- [ ] ドキュメント（ADR / API仕様）が更新されている
