## ユーザーストーリー

| 項目                   | 内容                                                       |
| ---------------------- | ---------------------------------------------------------- |
| **Who**（誰が）        | 管理者                                                     |
| **What**（何をしたい） | 登録ユーザーの一覧を閲覧したい                             |
| **Why**（なぜ・価値）  | 誰がサービスを使っているか把握して運用状況を確認したいから |

**ストーリー文（上記を文章化）:**

> **管理者** として、**登録ユーザーの一覧を閲覧したい**。なぜなら **誰がサービスを使っているか把握して運用状況を確認したい** からだ。

## 背景・補足

管理画面にてユーザー一覧（名前・ロール・最終ログイン日時）を表示する。一般ユーザーは管理画面へのアクセスを禁止し、403エラーを返す。

## 受け入れ基準

- [ ] **Given** 管理者  
       **When** 管理画面のユーザー一覧ページにアクセスする  
       **Then** 登録済みユーザーの一覧（名前・ロール・最終ログイン日時）が表示される

- [ ] **Given** 一般ユーザー  
       **When** 管理画面URLにアクセスする  
       **Then** 403エラーになる
