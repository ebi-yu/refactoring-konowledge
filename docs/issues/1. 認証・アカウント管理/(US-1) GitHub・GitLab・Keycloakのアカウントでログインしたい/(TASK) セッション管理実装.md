## 概要

ログイン後のセッション発行・検証・ログアウト処理の実装。

## 親 Story

- Story: (US-1) GitHub・GitLab・Keycloakのアカウントでログインしたい

## 作業内容

- [ ] OAuth コールバック成功後にサーバーサイドセッションを発行する
- [ ] セッション有効期限（例: 7日間）と自動延長ポリシーを設定する
- [ ] 認証済みセッションを検証するミドルウェアを実装する
- [ ] `/auth/logout` エンドポイントを実装しセッションを破棄する
- [ ] CSRF トークンの組み込み（セッションに紐付け）を実装する

## 技術メモ

- セッションストアは Redis または DB セッションテーブルを使用する
- `HttpOnly` / `Secure` / `SameSite=Lax` Cookie 属性を設定してセキュリティを確保する
- JWT ステートレスセッションと比較して、無効化が容易なサーバーサイドセッションを採用する
