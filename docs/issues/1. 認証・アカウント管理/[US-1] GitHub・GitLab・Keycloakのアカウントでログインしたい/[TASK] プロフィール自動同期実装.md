## 概要

OAuth プロバイダーから取得した表示名・アバター URL をユーザーレコードに同期する処理実装。

## 親 STORY

- STORY: [US-1] GitHub・GitLab・Keycloakのアカウントでログインしたい

## 作業内容

- [ ] ログイン / コールバック時にプロバイダーから取得した `display_name` と `avatar_url` を users テーブルに保存する
- [ ] プロフィールページで同期済みの表示名・アバターを表示するコンポーネントを実装する
- [ ] アバター URL が変更された場合に次回ログイン時に自動更新される仕組みを実装する
- [ ] アバターが取得できない場合のフォールバック表示（イニシャルアイコン等）を実装する

## 技術メモ

- アバター画像は外部 URL を直接参照する（アップロード不要）
- プライバシー配慮で avatar_url は外部に漏らさない（必要に応じてプロキシ経由にする）
