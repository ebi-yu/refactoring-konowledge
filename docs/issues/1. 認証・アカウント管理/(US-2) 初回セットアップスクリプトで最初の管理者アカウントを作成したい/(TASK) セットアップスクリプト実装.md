## 概要

初回起動時に管理者アカウントを作成する CLI スクリプト（`setup.sh` または `npm run setup`）の実装。

## 親 Story

- Story: (US-2) 初回セットアップスクリプトで最初の管理者アカウントを作成したい

## 作業内容

- [ ] `scripts/setup.sh`（または `npm run setup`）を作成する
- [ ] スクリプト実行時に管理者にするOAuthアカウント識別子（例: GitHub username）を引数または対話入力で受け取る
- [ ] 管理者が既に存在する場合はスキップして警告メッセージを表示する
- [ ] 実行結果（成功/スキップ）をコンソールに出力する
- [ ] README にセットアップ手順を記載する

## 技術メモ

- スクリプトは冪等に設計し、複数回実行しても副作用が出ないようにする
- DB 接続設定は環境変数（`.env`）から読み込む
- Docker 環境では `docker compose run app npm run setup` のような形で実行できるようにする
