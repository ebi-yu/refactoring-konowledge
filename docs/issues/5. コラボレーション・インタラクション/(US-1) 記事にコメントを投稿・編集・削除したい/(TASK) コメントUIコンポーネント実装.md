## 概要

コメント一覧・投稿フォーム・編集・削除ボタンの UI コンポーネント実装。

## 親 Story

- Story: (US-1) 記事にコメントを投稿・編集・削除したい

## 作業内容

- [ ] 記事詳細ページ下部にコメントセクションを追加する
- [ ] コメント一覧コンポーネントを実装する（投稿者アバター・名前・日時・本文）
- [ ] コメント投稿フォーム（テキストエリア・投稿ボタン）を実装する
- [ ] 自分のコメントに「編集」「削除」ボタンを表示する
- [ ] インライン編集（クリックで編集モードに切り替え）を実装する
- [ ] コメント削除時に確認ダイアログを表示する

## 技術メモ

- コメント数をリアルタイムで更新する（楽観的更新または Socket.io）
- 未ログインユーザーにはコメントフォームの代わりに「コメントするにはログインが必要です」を表示する

## 完了の定義（DoD）

- [ ] コードが実装されている
- [ ] ユニットテストが書かれ、CI が通過している
- [ ] セルフレビュー済み（lint / format エラーなし）
- [ ] 親 Story の担当者にレビュー依頼済み
