## 概要

記事作成フォームページの実装（タイトル入力・エディタ・投稿ボタン）。

## 親 Story

- Story: (US-1) タイトルとMarkdown本文で記事を作成したい

## 作業内容

- [ ] `/articles/new` ルートに記事作成ページを実装する
- [ ] タイトル入力フィールドを実装する
- [ ] Markdown エディタコンポーネントを配置する
- [ ] 公開範囲セレクターを配置する
- [ ] 「投稿」ボタンで API を呼び出し、成功後に記事詳細ページへリダイレクトする
- [ ] バリデーションエラー時にエラーメッセージを表示する

## 技術メモ

- フォームの状態管理には Vue の reactive または Pinia を使用する
- 未保存の変更がある状態でページを離れる際は離脱確認ダイアログを表示する

## 完了の定義（DoD）

- [ ] コードが実装されている
- [ ] ユニットテストが書かれ、CI が通過している
- [ ] セルフレビュー済み（lint / format エラーなし）
- [ ] 親 Story の担当者にレビュー依頼済み
