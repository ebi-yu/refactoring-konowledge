## 親 STORY

## 作業内容

- [ ] 記事詳細ページに「削除」ボタンを追加する（自分の記事の場合のみ表示）
- [ ] 削除ボタンクリック時に確認ダイアログを表示する（「本当に削除しますか？」）
- [ ] ダイアログで「OK」を選択した場合に `DELETE /api/articles/:id` を呼び出す
- [ ] 削除成功後に記事一覧ページにリダイレクトする
- [ ] 削除失敗時にエラートーストを表示する

## 技術メモ

- 確認ダイアログは共通コンポーネント（`ConfirmDialog`）として実装して他の削除操作でも再利用できるようにする
