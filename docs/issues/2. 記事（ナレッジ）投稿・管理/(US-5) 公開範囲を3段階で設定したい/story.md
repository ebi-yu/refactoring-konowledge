## ユーザーストーリー

| 項目                   | 内容                                                                        |
| ---------------------- | --------------------------------------------------------------------------- |
| **Who**（誰が）        | 投稿者                                                                      |
| **What**（何をしたい） | 記事を「公開 / ログインユーザーのみ / 非公開」の3段階で公開範囲を設定したい |
| **Why**（なぜ・価値）  | 社外秘の内容や書きかけの記事を誤って外部に見せたくないから                  |

**ストーリー文（上記を文章化）:**

> **投稿者** として、**記事を「公開 / ログインユーザーのみ / 非公開」の3段階で公開範囲を設定したい**。なぜなら **社外秘の内容や書きかけの記事を誤って外部に見せたくないから** だからだ。

## 背景・補足

記事の公開範囲を `public`（誰でも閲覧可）・`login_only`（ログイン済みのみ）・`private`（自分のみ）の3段階で設定できる。デフォルトは `login_only` とし、明示的に `public` を選んだ記事のみ外部公開する。

## 受け入れ基準

- [ ] **Given** 記事作成時  
       **When** 「ログインユーザーのみ」を選択して投稿する  
       **Then** 未ログインユーザーにはその記事が表示されない

- [ ] **Given** 「非公開」の記事  
       **When** 自分以外がアクセスする  
       **Then** 403エラーになる
