## ユーザーストーリー

| 項目                   | 内容                                                   |
| ---------------------- | ------------------------------------------------------ |
| **Who**（誰が）        | 投稿者                                                 |
| **What**（何をしたい） | 公開済み記事の内容を後から編集・更新したい             |
| **Why**（なぜ・価値）  | 状況が変わったときに正しい情報を読者に届け続けたいから |

**ストーリー文（上記を文章化）:**

> **投稿者** として、**公開済み記事の内容を後から編集・更新したい**。なぜなら **状況が変わったときに正しい情報を読者に届け続けたいから** だからだ。

## 背景・補足

記事の情報は時間とともに陳腐化する。投稿者が自分の記事を自由に編集できる仕組みを提供することで、常に最新の正確な情報を維持できる。他者の記事は編集できないよう権限チェックが必要。

## 受け入れ基準

- [ ] **Given** 自分の記事の投稿者  
       **When** 編集ページで内容を変更して更新ボタンを押す  
       **Then** 記事が更新されて詳細ページに変更が反映される

- [ ] **Given** 他人の記事  
       **When** 編集URLにアクセスする  
       **Then** 403エラーになる
