## 概要

既存記事データを読み込んで表示する編集フォームページの実装。

## 親 STORY

- STORY: [US-3] 公開済み記事を編集・更新したい

## 作業内容

- [ ] `/articles/:id/edit` ルートに記事編集ページを実装する
- [ ] ページ表示時に `GET /api/articles/:id` で既存記事データを取得してフォームに展開する
- [ ] タイトル・本文・公開範囲の各フィールドに現在の値を初期表示する
- [ ] 「更新」ボタンで `PUT /api/articles/:id` を呼び出し、成功後に記事詳細ページへリダイレクトする
- [ ] 自分以外の記事にアクセスした場合はエラーページを表示する

## 技術メモ

- 記事作成ページのフォームコンポーネントを再利用し、`mode: 'edit'` プロパティで挙動を切り替える
