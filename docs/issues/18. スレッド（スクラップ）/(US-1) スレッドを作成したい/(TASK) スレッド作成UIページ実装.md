## 概要

スレッドタイトル入力フォームとスレッド一覧ページの実装。

## 親 Story

- Story: (US-1) スレッドを作成したい

## 作業内容

- [ ] スレッド作成フォームページ（タイトル入力・公開設定）の実装
- [ ] スレッド一覧ページ（自分のスレッド + 公開スレッド）の実装
- [ ] スレッドカード（タイトル・作成日時・投稿数・公開状態バッジ）コンポーネントの実装
- [ ] 作成成功後のスレッド詳細ページへのリダイレクト処理

## 技術メモ

- スレッド一覧は最終更新日時の降順でソートする
- 自分のスレッドと他ユーザーの公開スレッドをタブで切り替えられると便利
- スレッドカードは簡潔にタイトルと直近の投稿スニペットを表示する
