## 概要

POST /api/v1/attachments エンドポイントの実装（マルチパートフォームデータとJSONレスポンス）。

## 親 Story

- Story: (US-2) REST APIでファイルアップロードをしたい

## 作業内容

- [ ] `POST /api/v1/attachments` エンドポイントの実装（multipart/form-data対応）
- [ ] アップロードファイルをクラウドストレージに保存する処理
- [ ] ファイルメタデータ（名前・サイズ・MIMEタイプ・URL）をDBに保存する処理
- [ ] レスポンスにファイルURLとIDを含むJSONを返す実装
- [ ] ファイルサイズ・MIMEタイプのバリデーション

## 技術メモ

- マルチパートパーサーは multer / busboy 等のライブラリを使用
- ファイルサイズ上限はデフォルト10MBとし設定可能にする
- 許可MIMEタイプはホワイトリスト方式で管理する
