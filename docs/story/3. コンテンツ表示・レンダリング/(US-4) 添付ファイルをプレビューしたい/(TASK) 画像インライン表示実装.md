## 概要

添付画像 URL を Markdown で `<img>` タグにレンダリングし、クリックで拡大表示するライトボックスの実装。

## 親 Story

- Story: (US-4) 添付ファイルをプレビューしたい

## 作業内容

- [ ] Markdown レンダラーで `![alt](url)` を `<img>` タグに変換して表示する
- [ ] 画像クリック時にライトボックス（拡大表示モーダル）を開くコンポーネントを実装する
- [ ] ライトボックスに前/次ナビゲーションボタンとキーボード操作（矢印キー・Escキー）を実装する
- [ ] 画像の遅延読み込み（`loading="lazy"`）を設定する
- [ ] 代替テキスト（`alt` 属性）を正しく設定する

## 技術メモ

- ライトボックスは既存ライブラリ（vue-easy-lightbox 等）を活用するか独自実装する
- 記事本文内の全画像をまとめてライトボックスのギャラリーに登録する
