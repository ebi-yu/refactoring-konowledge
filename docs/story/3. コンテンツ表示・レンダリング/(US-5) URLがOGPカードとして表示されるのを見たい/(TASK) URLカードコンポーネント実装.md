## 概要

OGP データを表示する URL カード Vue コンポーネントの実装。

## 親 Story

- Story: (US-5) URLがOGPカードとして表示されるのを見たい

## 作業内容

- [ ] `UrlCard` コンポーネントを実装する（`url` プロパティで URL を受け取る）
- [ ] コンポーネントマウント時に `/api/ogp?url=` を呼び出して OGP データを取得する
- [ ] タイトル・説明・OGP 画像・ドメイン名をカード形式で表示する
- [ ] OGP データがない場合は URL のみを表示するフォールバック UI を実装する
- [ ] Markdown レンダラーで URL のみの行を検出して `UrlCard` に置換する処理を実装する

## 技術メモ

- カード表示は記事本文の横幅に合わせたレスポンシブデザインにする
- OGP 画像の読み込みエラー時にフォールバックアイコンを表示する
