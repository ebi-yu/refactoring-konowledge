## 概要

管理画面のストレージ設定フォームページの実装（接続テストボタン含む）。

## 親 Story

- Story: (US-4) クラウドストレージに設定したい

## 作業内容

- [ ] ストレージ設定フォームページ（`/admin/settings/storage`）の実装
- [ ] プロバイダー選択（S3 / R2 / MinIO）によるフォームフィールドの動的切り替え
- [ ] 「接続テスト」ボタンとテスト結果の表示コンポーネント
- [ ] フォーム保存後の成功・失敗トースト通知

## 技術メモ

- secret_keyフィールドはパスワード型で表示・非表示を切り替え可能にする
- 接続テストは保存前にも実行できるようにしてUXを向上させる
- プロバイダーに応じて不要なフィールドを非表示にしてUIをシンプルにする
