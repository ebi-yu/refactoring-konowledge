## 作業内容

## 技術メモ

- 最後の管理者チェックは `SELECT COUNT(*) WHERE role = 'admin'` でカウントして1以下の場合はエラー
- ロール変更後はそのユーザーの既存セッションを再発行または無効化する
- バリデーションにはスキーマバリデーションライブラリを使用
