## 概要

記事作成・編集フォームに公開範囲選択 UI を追加。

## 親 Story

- Story: (US-5) 公開範囲を3段階で設定したい

## 作業内容

- [ ] 記事作成・編集フォームに公開範囲セレクター（ラジオボタンまたはドロップダウン）を追加する
- [ ] 選択肢：「公開」「ログインユーザーのみ」「非公開」を表示する
- [ ] デフォルト値を「ログインユーザーのみ」に設定する
- [ ] 各選択肢にアイコンと説明テキストを表示して意味を分かりやすくする
- [ ] フォーム送信時に選択した visibility 値を API リクエストに含める

## 技術メモ

- 選択肢の内部値は `public` / `login_only` / `private` の英語識別子を使用する
- 編集フォームでは既存の visibility 値をデフォルト選択状態にする

## 完了の定義（DoD）

- [ ] コードが実装されている
- [ ] ユニットテストが書かれ、CI が通過している
- [ ] セルフレビュー済み（lint / format エラーなし）
- [ ] 親 Story の担当者にレビュー依頼済み
