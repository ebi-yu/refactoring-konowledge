## 概要

S3/R2/MinIO 等のクラウドストレージへのファイル保存・取得・削除処理の実装。

## 親 Story

- Story: (US-7) ファイルを添付したい

## 作業内容

- [ ] ストレージプロバイダー（S3 / R2 / MinIO）の接続設定を環境変数から読み込む
- [ ] ファイルアップロード（`putObject`）・取得 URL 生成（`getSignedUrl`）・削除（`deleteObject`）の共通ストレージモジュールを実装する
- [ ] 記事削除時に関連する添付ファイルをストレージからも削除する処理を実装する
- [ ] プライベートファイルは署名付き URL で一時的にアクセス可能にする

## 技術メモ

- ローカル開発では MinIO を Docker コンテナで起動して S3 互換 API で動作させる
- `@aws-sdk/client-s3` を使用し、エンドポイントを環境変数で切り替えることで各プロバイダーに対応する
