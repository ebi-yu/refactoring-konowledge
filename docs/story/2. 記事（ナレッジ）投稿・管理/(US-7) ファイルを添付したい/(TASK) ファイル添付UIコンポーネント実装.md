## 概要

エディタへのドラッグ&ドロップ添付と Markdown リンク自動挿入コンポーネントの実装。

## 親 Story

- Story: (US-7) ファイルを添付したい

## 作業内容

- [ ] エディタ領域にドラッグ&ドロップイベントを追加する
- [ ] ドロップされたファイルを `POST /api/attachments` でアップロードする
- [ ] アップロード中はプログレス表示（スピナーやプログレスバー）を表示する
- [ ] アップロード完了後にカーソル位置に Markdown 記法（`![alt](url)` または `[filename](url)`）を自動挿入する
- [ ] クリップボードからの貼り付け（`paste` イベント）でも同様の処理を実装する

## 技術メモ

- ファイルアップロードは `FormData` を使用して `fetch` または `axios` で送信する
- 複数ファイルの同時ドロップに対応する
