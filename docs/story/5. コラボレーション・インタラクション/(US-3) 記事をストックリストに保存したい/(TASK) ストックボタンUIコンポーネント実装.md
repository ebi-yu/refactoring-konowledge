## 概要

記事詳細ページのストック追加・解除ボタンコンポーネントの実装。

## 親 Story

- Story: (US-3) 記事をストックリストに保存したい

## 作業内容

- [ ] 記事詳細ページにストックボタン（🔖アイコン + ストック数表示）を追加する
- [ ] ストック済みの場合はボタンをハイライト表示する
- [ ] ボタンクリックでストック追加・解除をトグルする処理を実装する
- [ ] 楽観的更新でストック操作を即時反映する
- [ ] 未ログインユーザーにはクリック時にログインページへ誘導する

## 技術メモ

- ストック数は記事詳細 API のレスポンスから取得する
- ストック状態は Pinia ストアで管理してコンポーネント間で共有する

## 完了の定義（DoD）

- [ ] コードが実装されている
- [ ] ユニットテストが書かれ、CI が通過している
- [ ] セルフレビュー済み（lint / format エラーなし）
- [ ] 親 Story の担当者にレビュー依頼済み
