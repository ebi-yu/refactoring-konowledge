## 概要

公開記事一覧・詳細ページでゲストアクセス時も正しく表示されるよう対応。

## 親 Story

- Story: (US-3) ログインせずに公開記事を閲覧したい

## 作業内容

- [ ] 記事一覧ページで未ログイン時にもページが正しく表示されることを確認・修正する
- [ ] 記事詳細ページで未ログイン時に `visibility=public` の記事のみ表示し、それ以外はリダイレクトまたはエラーページを表示する
- [ ] ヘッダー等のナビゲーションコンポーネントで未ログイン時は「ログイン」ボタンを表示する
- [ ] SSR（サーバーサイドレンダリング）使用時にゲストセッションで正しくデータフェッチされることを確認する

## 技術メモ

- 記事取得 API（`GET /api/articles/:id`）は visibility=public の場合は認証なしで 200 を返す
- ゲスト向けにはコメント投稿・ストック等のアクション UI を非表示または無効化する

## 完了の定義（DoD）

- [ ] コードが実装されている
- [ ] ユニットテストが書かれ、CI が通過している
- [ ] セルフレビュー済み（lint / format エラーなし）
- [ ] 親 Story の担当者にレビュー依頼済み
