# [Epic] Chrome拡張（クイック投稿）

## 概要

ブラウザ上でその場でスレッドにメモを投稿できる Chrome 拡張。
調べ物中に気づいたこと・参考にした URL・断片的なメモをその場で記録でき、後から整理する。
投稿先は Epic 18 のスレッド（スクラップ）機能。拡張はクライアントに過ぎず、データはサーバー側のスレッドとして管理される。
**全機能が Ph2 スコープ。**

## Epic ストーリー

**投稿者** として、**ブラウザで調べ物をしながらその場でメモをスレッドに投稿したい**。なぜなら **気づいたことをわざわざアプリを開いて書くのは手間がかかり、そのうちに忘れてしまうから**。

## スコープ（Story 一覧）

> 優先度: `Must` = リリース必須 / `Should` = できれば含める / `Could` = 余裕があれば
> **⚠️ このEpicは Ph2 スコープのため Ph1 リリースには含まれない**

| 優先度 | # | ストーリー（一文） | Issue |
|--------|---|-------------------|-------|
| `Must` | 1 | 投稿者として、ブラウザのツールバーからメモをスレッドに素早く投稿したい。なぜなら調べ物中に気づいたことをその場で記録して、後から参照できるようにしたいから。 | - |
| `Must` | 2 | 投稿者として、Chrome拡張から投稿先のスレッドを選択・新規作成したい。なぜならテーマごとにメモをまとめて管理したいから。 | - |
| `Could` | 3 | 投稿者として、選択したテキストを右クリックメニューから直接スレッドに投稿したい。なぜならページ内の気になる文章を手間なくメモしたいから。 | - |

## スコープ外

- **スレッド機能本体**: Epic 18 で対応。拡張は投稿クライアントのみ
- **Firefox / Safari 対応**: Chrome のみ。他ブラウザは将来対応
