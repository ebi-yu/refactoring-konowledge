## 技術メモ

- `tsgo` は TypeScript の高速型チェックツール（`typescript-go`）
- `--noEmit` フラグにより、出力ファイルを生成せず型チェックのみ実行する
- 型エラー修正時は `any` の安易な使用を避け、適切な型定義を行うこと
